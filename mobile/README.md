# University Room Booking Mobile

Mobile client scaffold for the University Room Booking System.

## Setup

```bash
npm install
```

## Run

```bash
npx expo start
```

## Structure

- `app/`: Expo Router screens and layouts
- `services/`: API services
- `components/`: shared UI components
- `hooks/`: custom hooks
- `utils/`: helper functions
- `types/`: shared TypeScript types

Empty folders are kept with `.gitkeep` so the team can follow one structure.
